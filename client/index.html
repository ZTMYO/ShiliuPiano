关于 石榴钢琴

多人实时在线钢琴弹奏应用。一人弹奏，多人实时收听。支持键盘弹奏、触摸滑键、MIDI 设备连接和弹幕聊天。

点击右上角的MIDI按钮连接外接 MIDI 键盘进行演奏。

点击左上角的我要弹奏按钮开始弹奏，其他人可实时收听。

钢琴键盘右上角的键位显示按钮可切换琴键对应的键盘键位。

弹奏时背景中的可视化红绿光球会向中心靠拢，停止后散开。

前端：Vite + Vanilla JavaScript + Canvas
后端：Node.js + Express + Socket.IO

钢琴音源及部分代码灵感来自 AutoPiano
